深色主题设计实践：不只是把背景变黑

2026年4月30日·CSS·预计阅读时间
暗黑模式CSS 变量视觉设计

深色主题并不是简单地把背景设置为纯黑、文字设置为纯白。真正舒适的暗黑界面需要层级、对比、留白和状态反馈。

使用 CSS 变量管理主题

本网站把背景、卡片、文字、边框、强调色等抽象为 CSS 变量，再通过给 body 添加 light-theme 类实现主题切换。

:root {
  --bg: #050505;
  --surface: #101010;
  --text: #f4f4f5;
}

暗黑界面的层级

深色页面中，卡片不应该和背景完全融为一体。可以通过略亮的 surface、细边框和轻微阴影建立层级。

交互细节

  • hover 不宜过度放大，轻微上浮即可。
  • focus 状态必须清晰，方便键盘用户使用。
  • 浅色主题下不能使用过低透明度的文字。

好的深色主题应该让内容更突出，而不是让用户努力寻找内容。
