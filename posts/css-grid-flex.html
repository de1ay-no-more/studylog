CSS Flex 与 Grid：什么时候该用哪一个？

2026年5月26日·CSS·预计阅读时间
CSS3FlexGrid响应式

Flex 和 Grid 都是现代 CSS 布局的核心工具。它们并不是互相替代的关系，而是分别擅长不同维度的布局问题。

Flex：适合一维布局

当元素主要沿着一个方向排列时，Flex 更直接。例如导航菜单、按钮组、卡片内部的图文排列都适合使用 Flex。

.nav-links {
  display: flex;
  align-items: center;
  gap: 8px;
}

Grid：适合二维布局

当布局同时涉及行和列时，Grid 更稳定。例如项目展示页、博客卡片列表、仪表盘模块都适合使用 Grid。

.project-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 22px;
}

选择方法

  • 只关心横向或纵向排列：优先 Flex。
  • 需要控制多列、多行和整体网格：优先 Grid。
  • 实际项目中两者可以组合使用：外层 Grid，内层 Flex。

本网站项目页使用 Grid 控制整体卡片布局，卡片内部的标签行则用 Flex 自动换行。
