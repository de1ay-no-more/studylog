# StudyLog 个人技术博客网站项目说明

## 项目主题

本项目以“个人技术博客网站”为主题，设计并实现了一个暗黑极简风格的静态技术博客。网站主要使用 HTML5、CSS3 和原生 JavaScript，无后端依赖，可直接在浏览器中打开运行。

## 页面结构

- `index.html`：首页，包含导航栏、个人简介、头像、最新 3 篇博客摘要和 Footer。
- `blog.html`：博客列表页，展示所有文章，支持分类筛选、关键词搜索和日期升降序排序。
- `posts/*.html`：文章详情页，采用独立 HTML 页面方式实现。
- `about.html`：个人信息页，包含个人介绍、联系方式、技能展示和留言板。
- `projects.html`：项目展示页，使用 Grid 网格展示课程项目和前端练习。
- `404.html`：自定义 404 页面，作为自由发挥功能。

## 文章详情实现方案

本项目选择“独立 HTML 页面”方案实现文章详情页。每篇文章在 `posts/` 文件夹中有一个独立页面，例如：

- `posts/html-semantic.html`
- `posts/css-grid-flex.html`
- `posts/js-localstorage.html`

博客列表页中的文章卡片通过 `assets/js/data.js` 中的文章元数据生成，点击后跳转到对应的独立文章页面。该方案结构直观、便于教师查看，也符合静态网站提交要求。

## 主要功能

1. 首页模块
   - 顶部导航栏与当前页面高亮
   - 个人简介、头像、昵称和方向介绍
   - 最新 3 篇博客摘要卡片
   - 统一 Footer
   - 打字机效果和渐入动画

2. 博客模块
   - 所有文章卡片展示
   - 分类标签筛选
   - 关键词搜索标题、摘要和标签
   - 日期升序/降序排序
   - 独立文章详情页
   - 阅读进度条和预计阅读时间

3. 个人信息模块
   - 个人介绍和联系方式
   - 技能进度展示
   - 简易留言板
   - 表单空内容与长度验证
   - 使用 localStorage 持久化留言数据
   - 用户输入内容转义后渲染，避免插入 HTML

4. 项目展示模块
   - CSS Grid 响应式项目卡片
   - 项目名称、简介、技术标签和封面图
   - 鼠标悬停上浮、阴影和图片缩放效果

5. 自由发挥功能
   - 深色/浅色主题切换
   - 主题偏好 localStorage 保存
   - 文章阅读进度条
   - 预计阅读时间
   - 自定义 404 页面
   - 页面进入动画与卡片悬停动画

## 技术亮点

- 使用语义化 HTML5 标签组织页面结构。
- 使用 CSS 变量统一管理暗黑/浅色主题。
- 使用 CSS Grid 和 Flex 实现响应式布局。
- 使用原生 JavaScript 完成筛选、搜索、排序、主题切换和留言板。
- 使用 localStorage 保存用户主题偏好和留言数据。
- 关注可访问性：表单 label、focus 状态、alt 文本、触控尺寸和 reduced motion。

## 运行方式

直接用浏览器打开 `index.html` 即可，也可以使用 VS Code Live Server 插件预览。
