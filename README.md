# StudyLog 个人技术博客

暗黑极简风格的 HTML5 + CSS3 + JavaScript 静态个人技术博客。

## 入口

打开 `index.html`。

## 页面

- `index.html` 首页
- `blog.html` 博客列表
- `posts/*.html` 独立文章详情页
- `about.html` 个人信息与留言板
- `projects.html` 项目展示
- `404.html` 自定义 404

## 功能

- 当前页面导航高亮
- 最新文章展示
- 分类筛选、搜索、排序
- 独立文章详情页
- 阅读进度条、预计阅读时间
- 深浅主题切换并持久化
- localStorage 留言板
- 项目 Grid 响应式展示
